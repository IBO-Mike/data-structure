# 数据结构实验 - 线性表及其应用

## 实验内容

本实验实现了线性表的两种存储结构（顺序存储和链式存储），并应用这些结构实现了一元多项式的运算和长整数的加减运算。

### 主要功能

1. **线性表基本操作的实现**
   - 顺序表（数组实现）
   - 链表（单链表实现）

2. **一元多项式运算**
   - 多项式的存储（只存储非零项）
   - 多项式加法
   - 多项式减法
   - 多项式乘法
   - 多项式求值

3. **长整数运算（可选内容）**
   - 支持 100 位以内的长整数
   - 长整数加法
   - 长整数减法
   - 支持负数运算

## 核心数据结构设计

### 1. 顺序表 (LinearListArray)

使用动态数组实现的线性表，支持自动扩容。

**特点：**
- 随机访问，时间复杂度 O(1)
- 插入删除需要移动元素，时间复杂度 O(n)
- 预分配空间，按需扩容

**基本操作：**
- `insert(pos, value)` - 在位置 pos 插入元素
- `remove(pos, value)` - 删除位置 pos 的元素
- `getValue(pos)` - 获取位置 pos 的元素
- `indexOf(value)` - 查找值为 value 的位置
- `append(value)` - 在末尾追加元素

### 2. 链表 (LinearListLinked)

使用单链表实现的线性表。

**特点：**
- 插入删除不需要移动元素，时间复杂度 O(1)（已知位置）
- 不能随机访问，时间复杂度 O(n)
- 动态分配内存，无需预先分配

**结点结构：**
```cpp
struct Node {
    int data;      // 数据域
    Node* next;    // 指针域
};
```

### 3. 多项式 (Polynomial)

使用链表存储一元多项式，只存储系数非零的项。

**特点：**
- 节省存储空间（不存储系数为零的项）
- 按指数降序排列
- 自动合并同类项

**结点结构：**
```cpp
struct PolyNode {
    int coefficient;  // 系数
    int exponent;     // 指数
    PolyNode* next;   // 指针域
};
```

**示例：**
- P₁(x) = 3x⁴ + 2x² + 5
- P₂(x) = 4x³ - x² + 2

**运算：**
- 加法：P₁(x) + P₂(x)
- 减法：P₁(x) - P₂(x)
- 乘法：P₁(x) × P₂(x)
- 求值：P(x) 在给定 x 时的值

### 4. 长整数 (BigInt)

使用链表实现任意精度的整数运算（支持 100 位以内）。

**特点：**
- 每个结点存储一位数字
- 支持正负数
- 低位在前，高位在后（便于计算）

**结点结构：**
```cpp
struct BigIntNode {
    int digit;        // 一位数字 (0-9)
    BigIntNode* next; // 指向下一位
};
```

**示例：**
- num1 = 123456789012345678901234567890
- num2 = 987654321098765432109876543210
- 支持超大数运算（100 位）

## 算法复杂度分析

### 顺序表 vs 链表

| 操作 | 顺序表 | 链表 |
|------|--------|------|
| 访问 | O(1) | O(n) |
| 插入（已知位置） | O(n) | O(1) |
| 删除（已知位置） | O(n) | O(1) |
| 查找 | O(n) | O(n) |
| 空间效率 | 可能浪费空间 | 按需分配 |

### 多项式运算

设两个多项式的项数分别为 m 和 n：

| 运算 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 加法 | O(m + n) | O(m + n) |
| 减法 | O(m + n) | O(m + n) |
| 乘法 | O(m × n) | O(m × n) |

### 长整数运算

设两个数的位数分别为 m 和 n：

| 运算 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 加法 | O(max(m, n)) | O(max(m, n)) |
| 减法 | O(max(m, n)) | O(max(m, n)) |

## 关键实现细节

### 1. 多项式的有序存储

多项式按指数降序存储，这样在进行加减运算时可以采用归并的思想，提高效率。

```cpp
// 添加项时保持有序
void Polynomial::addTerm(int coefficient, int exponent) {
    if (coefficient == 0) return;  // 不添加系数为 0 的项
    
    // 查找是否已有相同指数的项
    PolyNode* current = head;
    while (current != nullptr) {
        if (current->exponent == exponent) {
            // 合并同类项
            current->coefficient += coefficient;
            if (current->coefficient == 0) {
                // 如果系数变为 0，删除该项
                removeValue(current->coefficient - coefficient);
            }
            return;
        }
        current = current->next;
    }
    
    // ... 插入到合适位置
}
```

### 2. 长整数的链表反转

为了从低位到高位进行计算，需要在计算前反转链表，计算后再反转回来。

```cpp
auto reverseList = [](BigIntNode* node) -> BigIntNode* {
    BigIntNode* prev = nullptr;
    BigIntNode* current = node;
    while (current != nullptr) {
        BigIntNode* next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    return prev;
};
```

### 3. 自动扩容机制

顺序表在空间不足时自动扩容为原来的 2 倍。

```cpp
void LinearListArray::expand(int newCapacity) {
    if (newCapacity <= capacity) return;
    
    int* newData = new int[newCapacity];
    for (int i = 0; i < length; ++i) {
        newData[i] = data[i];
    }
    delete[] data;
    data = newData;
    capacity = newCapacity;
}
```
